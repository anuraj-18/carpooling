how is our work

process flow

  • User Registration
  • Make Request
  • Receive Response
  • Car Pool

This is the first step The Registration Process. The user has to fill in his details and he will receive a mail containing his confidential security code and password which he can use in the future to login.

The user enters the details of the travel which includes the location, the time and the date on which he wants to go. This sends a request on our database and we then process the request.

Once the request is processed the user can login using his credentials and can see weather his request matches with that of some other person. There the email and phone number will be mentioned so both of them can talk to each other.

Now it's time to save Money and save Gas. It's time to Car-Pool. We at the CarPoolers thank you for using our service and for your contribution to the environment by causing less polution.
